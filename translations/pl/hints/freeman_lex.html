Nie mam pojęcia, jak zacząć rozwiązywać tę misję.

Proszę złapać pomysł 💡! Proszę zadeklarować zmienne do przechowywania najdłuższego słowa i jego długości. Podzielić zdanie na osobne słowa, przyjrzeć się każdemu z nich, znaleźć jego długość i porównać z zapamiętaną. Jeśli długość bieżącego słowa jest większa - proszę zaktualizować wartości zmiennych dla bieżącego najdłuższego słowa i jego długości.

Ok, od czego powinienem zacząć?

Proszę zadeklarować dwie zmienne: dla najdłuższego słowa z wartością początkową pustego ciągu; dla jego długości z wartością początkową 0. Na końcu ma zostać zwrócone najdłuższe słowo. 

res = ""
l = 0
...
return res
Co robić dalej?

Podzielmy zdanie na osobne słowa za pomocą metody str.split() bez argumentów i iterujmy po tych słowach za pomocą pętli for. 

for word in sentence.split():
    ...
Co zrobić z każdym słowem?

Ponieważ szukają Państwo najdłuższego słowa, należy obliczyć jego długość za pomocą funkcji len(). 

l1 = len(word)

Teraz, jeśli obliczona wartość l1 jest ściśle większa niż zapisana wartość l, należy zaktualizować zmienne res i l o nowe wartości word i l1. 

if l1> l: res, l = word, l1

Używane jest tutaj rozpakowywanie krotek. Ścisłe porównanie jest konieczne, aby zachować pierwsze słowo o najdłuższej długości.

Jak więc powinno wyglądać moje ostateczne rozwiązanie?

Wystarczy połączyć wszystkie poprzednie podpowiedzi i zawinąć kod w wyrażenie funkcyjne. Używanie podpowiedzi typu dla danych wejściowych i wyjściowych jest wysoce zalecane.

Poniżej znajduje się całe rozwiązanie:

def longest_word(sentence: str) -> str:
    
    res = ""
    l = 0 
    for word in sentence.split():
        l1 = len(word)
        if l1> l: res, l = word, l1
        
    return res

Proszę podzielić linię i wziąć maksimum według długości i odwróconego indeksu w linii. 

def longest_word(line: str) -> str:

    return line and max(line.split(), key=lambda w: (len(w), -line.index(w)))
