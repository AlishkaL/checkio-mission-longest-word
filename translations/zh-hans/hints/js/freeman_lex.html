让我们用string.split() 方法（不带参数）将句子分割成单词，然后用for 循环遍历这些单词。 

for (let word of sentence.split(" ")) {

如何处理每个单词？

由于您要查找最长的单词，因此需要使用string.length 属性计算长度。 

let l1 = word.length;

现在，如果计算值l1 严格大于存储值l ，则需要用word 和l1 的新值更新变量res 和l 。 

if (l1> l) [res, l] = [word, l1];

这里使用的是元组解包。需要进行严格比较，以保留长度最长的第一个字。

那么，我的最终解决方案应该是怎样的呢？

您只需将之前所有的提示连接起来，然后将代码封装成函数表达式即可。强烈建议对输入和输出数据使用类型提示。

以下是整个解决方案：

function longestWord(sentence: string): string {

    let res = "", l = 0;
    for (let word of sentence.split(" ")) {
        let l1 = word.length;
        if (l1> l) [res, l] = [word, l1];
        }
    return res;
}

使用特定函数分割该行并将其还原为单一值。 

function longestWord(line: string): string {

    return line.split(' ').reduce((pre, cur) => cur.length> pre.length? cur: pre, '');
}
