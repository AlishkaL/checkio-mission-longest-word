如何处理每个单词？

由于您要查找最长的单词，因此需要使用len() 函数计算长度。

现在，如果计算值l1 严格大于存储值l ，则需要用word 和l1 的新值更新变量res 和l 。

您只需将之前所有的提示连接起来，然后将代码封装成函数表达式即可。强烈建议对输入和输出数据使用类型提示。

以下是整个解决方案：

def longest_word(sentence: str) -> str:
    
    res = ""
    l = 0 
    for word in sentence.split():
        l1 = len(word)
        if l1> l: res, l = word, l1
        
    return res

分行，按长度和行中的反向索引取最大值。 

def longest_word(line: str) -> str:

    return line and max(line.split(), key=lambda w: (len(w), -line.index(w)))
