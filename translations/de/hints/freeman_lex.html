Haben Sie eine Idee 💡! Deklarieren Sie Variablen, um ein aktuelles längstes Wort und seine Länge zu speichern. Teilen Sie den Satz in einzelne Wörter auf, schauen Sie sich jedes einzelne an, ermitteln Sie seine Länge und vergleichen Sie sie mit der gespeicherten Länge. Wenn die Länge des aktuellen Wortes länger ist - aktualisieren Sie die Werte der Variablen für das aktuell längste Wort und seine Länge.

Ok, womit soll ich anfangen?

Deklarieren Sie zwei Variablen: für das längste Wort mit dem Anfangswert eines leeren Strings; für seine Länge mit dem Anfangswert 0. Sie werden am Ende das längste Wort zurückgeben. 

res = ""
l = 0
...
return res
Was ist als nächstes zu tun?

Zerlegen wir den Satz in einzelne Wörter mit der Methode str.split() ohne Argumente und iterieren wir über diese Wörter mit der Schleife for. 

for word in sentence.split():
    ...
Was tun wir mit jedem Wort?

Da Sie nach dem längsten Wort suchen, müssen Sie die Länge mit der Funktion len() berechnen. 

l1 = len(word)

Wenn nun der berechnete Wert l1 deutlich größer ist als der gespeicherte Wert l, müssen Sie die Variablen res und l mit den neuen Werten word und l1 aktualisieren. 

if l1> l: res, l = word, l1

Hier wird das Tupel-Entpacken verwendet. Ein strenger Vergleich ist erforderlich, um das erste Wort mit der größten Länge zu erhalten.

Wie sollte also meine endgültige Lösung aussehen?

Sie müssen nur alle vorherigen Hinweise zusammenführen und Ihren Code in einen Funktionsausdruck verpacken. Die Verwendung von Typ-Hinweisen für Eingabe- und Ausgabedaten wird dringend empfohlen.

Hier ist die gesamte Lösung:

def longest_word(sentence: str) -> str:
    
    res = ""
    l = 0 
    for word in sentence.split():
        l1 = len(word)
        if l1> l: res, l = word, l1
        
    return res

Teilen Sie die Zeile und nehmen Sie das Maximum nach Länge und umgekehrtem Index in der Zeile. 

def longest_word(line: str) -> str:

    return line and max(line.split(), key=lambda w: (len(w), -line.index(w)))
