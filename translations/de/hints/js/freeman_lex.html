Fangen Sie eine Idee 💡! Teilen Sie den Satz in einzelne Wörter auf, iterieren Sie über sie und speichern Sie das längste bei jedem Schritt in einer vordefinierten Variablen, geben Sie den Variablenwert am Ende zurück.

Ok, womit soll ich anfangen?

Deklarieren Sie zwei Variablen: für das längste Wort mit dem Anfangswert eines leeren Strings; für seine Länge mit dem Anfangswert 0. Sie werden am Ende das längste Wort zurückgeben. 

let res = "", l = 0;
...
return res;
Was ist als nächstes zu tun?

Zerlegen wir den Satz in einzelne Wörter mit der Methode string.split() ohne Argumente und iterieren wir über diese Wörter mit der Schleife for. 

for (let word of sentence.split(" ")) {
    ...
    }
Was tun wir mit jedem Wort?

Da Sie nach dem längsten Wort suchen, müssen Sie die Länge mit der Eigenschaft string.length berechnen. 

let l1 = word.length;

Wenn nun dieser berechnete Wert l1 deutlich größer ist als der gespeicherte Wert l, müssen Sie die Variablen res und l mit den neuen Werten word und l1 aktualisieren. 

if (l1> l) [res, l] = [word, l1];

Hier wird das Tupel-Entpacken verwendet. Ein strenger Vergleich ist erforderlich, um das erste Wort mit der größten Länge zu erhalten.

Wie sollte also meine endgültige Lösung aussehen?

Sie müssen nur alle vorherigen Hinweise zusammenführen und Ihren Code in einen Funktionsausdruck verpacken. Die Verwendung von Typ-Hinweisen für Eingabe- und Ausgabedaten wird dringend empfohlen.

Hier ist die gesamte Lösung:

function longestWord(sentence: string): string {

    let res = "", l = 0;
    for (let word of sentence.split(" ")) {
        let l1 = word.length;
        if (l1> l) [res, l] = [word, l1];
        }
    return res;
}

Klar, schnappen Sie es sich!

Teilen Sie die Zeile und reduzieren Sie sie mit einer speziellen Funktion auf einen einzigen Wert. 

function longestWord(line: string): string {

    return line.split(' ').reduce((pre, cur) => cur.length> pre.length? cur: pre, '');
}
