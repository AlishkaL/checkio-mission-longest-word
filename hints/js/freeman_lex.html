What to do with every word?

Since you are looking for the longest one, you need to calculate the length with string.length property. 

let l1 = word.length;

Now, if this calculated value l1 is strictly bigger then the stored one l, you need to update variables res and l with new values of word and l1. 

if (l1 > l) [res, l] = [word, l1];

Tuple unpacking is used here. Strict comparison is needed to keep the first word of longest length.

So, how my final solution should look like?

You just need to join all previous hints and wrap your code into function expression. Using type hints for input and output data are highly recommended.

Here is the whole solution:

function longestWord(sentence: string): string {

    let res = "", l = 0;
    for (let word of sentence.split(" ")) {
        let l1 = word.length;
        if (l1 > l) [res, l] = [word, l1];
        }
    return res;
}

Split the line and reduce it to single value using specific function. 

function longestWord(line: string): string {

    return line.split(' ').reduce((pre, cur) => cur.length > pre.length ? cur : pre, '');
}
